# Easy Commit
✉️ Git Commit Message Helper

![preview](./preview.png)

## Install

- Require
  - Node.js `>=10.3.0`
  - Git

```bash
npm install -g easy-commit
```

## Usage

Just enter `ec` command

```bash
ec
```

- Type commit message subject
- Type commit message body
- `Ctrl + C` to confirm
